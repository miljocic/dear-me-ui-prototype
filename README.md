# Dear Me - Mental Wellness App UI Prototype

A mobile-first UI prototype for a mental wellness companion app called **Dear Me**, designed to help users track their mental health and build positive habits. The project went through two stages: an initial collaborative Figma prototype, followed by an individual HTML/CSS implementation.

---

## Tech Stack

- **Figma** — collaborative hi-fi prototype (Stage 1)
- **HTML5 / CSS3** — structure and styling (Stage 2)
- **Bootstrap 5.3** — responsive layout and components
- **Bootstrap Icons** — iconography
- **Google Fonts (Fredoka)** — warm, friendly typography
- **Custom CSS variables** — consistent warm peach/teal/cream color palette

### Stage 2 — HTML/CSS Implementation (Milica Jocic)

| File | Screen | Description |
|------|--------|-------------|
| `homepage.html` | Home | Main dashboard with mascot, motivational quote, and navigation to core features (Meditate, Breathe, Journal, Goals, Calm me) |
| `feeling.html` | Mood Check-in | Emotion selection grid (6 moods with emojis) and free-text journal entry |
| `goal.html` | Add a Goal | Goal creation form with save functionality |
| `community.html` | Community | Community feed where users share thoughts and reply to each other |

## Design

- **Color palette:** warm peach (`#F4A261`), burnt coral (`#E76F51`), cream (`#FFF4E6`), mint teal (`#A8D5BA`)
- **Style:** soft rounded corners, card-based layout, drop shadows
- **Navigation:** persistent bottom nav bar (Profile, Calendar, Stats, Community)
- **Custom mascot** character included on homepage
